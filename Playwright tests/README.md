# Playwright Автотест для Википедии

Этот проект содержит автотест на Playwright для проверки статьи о Земле в Википедии.

## Описание теста

Тест выполняет следующие действия:
1. Открывает браузер Chrome
2. Переходит на главную страницу русской Википедии
3. Вводит "Земля" в поле поиска
4. Переходит на статью о Земле
5. Находит раздел "Химический состав атмосферы"
6. Проверяет, что в этом разделе содержится "20,95 % кислорода"

## Установка

```bash
npm install
```

## Установка браузеров Playwright

```bash
npx playwright install chromium
```

## Запуск тестов

```bash
# Запуск тестов в headless режиме
npm test

# Запуск тестов с видимым браузером
npm run test:headed

# Запуск тестов в режиме отладки
npm run test:debug
```

## Структура проекта

- `earth-test.spec.js` - основной тест для проверки статьи о Земле
- `playwright.config.js` - конфигурация Playwright
- `package.json` - зависимости проекта
